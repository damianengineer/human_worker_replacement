References

Every statistic, study, or quote on this site has a small source marker next to it. Click one and it brings you here, to that source's full entry — not straight out to the original. From here, each entry's own link goes to the actual study, article, or dataset, so you can always see exactly where a number came from before deciding whether to trust it. If a citation is ever incomplete or its details haven't been independently confirmed, that's labeled plainly rather than presented as more solid than it is.
